# 複数のエディター

次の例には、Tiptap の 3つの異なるインスタンスがあります。 1つ目はテキストの単一段落、2つ目はタスクリスト、3つ目はテキストを含むように構成されています。 それらはすべて単一の Y.js ドキュメントに保存され、他のユーザーとリアルタイムで同期できます。
